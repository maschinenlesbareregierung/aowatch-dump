# @malereg/aowatch-dump

## Installation

```
npm install @malereg/aowatch-dump -g
```

## Usage 

```
aowatch-dump -h
Usage: aowatch-dump [options]

Dumps data from Abgeordnetenwatch.de

Options:
  -t, --type <type>                which datatype to export (politician, election, location, committee, parliament, vote) (default: "location")
  -p, --path <path>                path to export data to (default: "/home/politik/projects/aowatch-dump")
  -c, --concurrency <concurrency>  how many parallel requests to use (default: 2)
  -h, --help                       output usage information
```
